Triggers & Output Options

1. Triggers (* and ?)

:*:btw Immediate (No Space needed).
:?:al Inside word (type "practical").

2. Output Control (B0 and O)

:B0O:<em> No Backspace, Omit space. Keeps <em>, adds closing tag, caret inside.
:O:addr Omits End Char. Type "addr" + Space. Result has NO space.

3. Capitalization (C, C1)

::twa Default (Smart):
twa → Taiwan
Twa → Taiwan
TWA → TAIWAN
:C:AHK Case Sensitive: Only triggers on CAPS.
:C1:im No Conformity:
Type "Im" → "iM" (Exact replacement).

4. Regex Triggers (JS RegExp)

/(\d+)x(\d+)/ Calculates product.
Type 10x10 or 5x4.
